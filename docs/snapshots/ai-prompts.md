# Cowork Forge 演示页面 AI 生图提示词

## 第 1 页：项目概览 (Overview)

**页面主题**: AI 驱动的智能研发平台 - 重新定义软件开发

**视觉风格要求**:
- 现代科技感插画风格，明亮清爽
- 扁平化设计 + 适度的渐变光晕效果
- 白色/浅蓝色为主色调，营造专业且有科技感的氛围
- 16:9 宽屏比例，适合商务汇报

**核心视觉元素**:
1. **中心图形**: 一个抽象的 AI 智能核心，环绕着 7 个彩色节点（代表 7 个阶段），使用圆形轨道连接
2. **7 个阶段节点**: 从左上开始顺时针排列，每个节点用不同颜色和图标表示
   - 💡 IDEA (青色)
   - 📋 PRD (紫色)
   - 🎨 DESIGN (粉色)
   - 📝 PLAN (橙色)
   - ⚙️ CODING (绿色)
   - 🔍 CHECK (红色)
   - 🚀 DELIVERY (蓝色)
3. **右侧数据指标**: 4 个大号数字展示牌
   - 10x 开发效率提升
   - 70% 人力成本降低
   - 7 自动化阶段
   - 99% 代码质量

**背景元素**:
- 柔和的蓝紫色渐变光晕
- 极简的几何线条装饰

**配色方案**:
- 主色: 白色 #FFFFFF
- 背景: 浅蓝 #F0F4FF
- 强调色: 蓝色 #3B82F6, 青色 #06B6D4, 绿色 #10B981, 橙色 #F97316, 紫色 #8B5CF6, 粉色 #EC4899

**中文提示词示例**:
> 现代科技感商业演示幻灯片，16:9宽屏，白色浅蓝色调，明亮清爽。一个抽象的AI智能核心圆形图标位于中央，环绕着7个彩色渐变圆形节点，形成圆形轨道，节点分别用简洁图标表示：灯泡、文档、设计画笔、代码符号、放大镜、火箭。右侧4个简约的数据展示卡片，大字号数字：10x、70%、7、99%。背景有柔和的蓝紫色渐变光晕和极简几何线条。扁平化设计风格，现代互联网公司风格，干净专业，商务科技感 --ar 16:9 --v 6

## 第 2 页：技术架构 (Architecture)

**页面主题**: 系统技术架构 - 分层解耦、模块化设计

**视觉风格要求**:
- 架构图风格，清晰的层级结构
- 从上到下的分层布局
- 圆角矩形卡片 + 连接线
- 明亮活泼的渐变色区分不同层级

**核心视觉元素**:
1. **交互层** (顶部，灰色调)
   - CLI 命令行
   - Web UI
   - Desktop 桌面端
   - API 接口

2. **流水线层** (蓝色渐变)
   - Pipeline Executor
   - Stage Dispatcher
   - Execution Context

3. **智能体层** (紫色渐变)
   - Iterative Assistant
   - Stage Agent
   - Human-in-Loop

4. **工具层** (青色渐变)，横向排列 6 个工具
   - 文件操作
   - 命令执行
   - 代码分析
   - 验证工具
   - 知识库
   - 部署

5. **基础设施层** (绿色渐变)
   - LLM Provider
   - File System
   - Project Runtime
   - Persistence

**布局要求**:
- 左侧 1/4: 技术栈列表 (Rust, Tauri, React, LLM API, Tokio)
- 右侧 3/4: 架构层级图
- 底部: 4 个架构原则标签

**配色方案**:
- 交互层: #F1F5F9 浅灰
- 流水线层: #3B82F6 → #60A5FA 蓝色渐变
- 智能体层: #8B5CF6 → #A78BFA 紫色渐变
- 工具层: #06B6D4 → #22D3EE 青色渐变
- 基础设施层: #10B981 → #34D399 绿色渐变

**中文提示词示例**:
> 现代技术架构图演示幻灯片，16:9宽屏，白色背景。从上到下的分层架构：顶层4个浅灰色圆角矩形（CLI、Web UI、Desktop、API），第二层3个蓝色渐变矩形（Pipeline Executor、Stage Dispatcher、Execution Context），第三层3个紫色渐变矩形（Iterative Assistant、Stage Agent、Human-in-Loop），第四层6个青色渐变小矩形横向排列（文件、命令、分析、验证、知识、部署），底层4个绿色渐变矩形（LLM Provider、File System、Runtime、Persistence）。各层之间用虚线连接。左侧有技术栈列表。底部4个简约标签：分层解耦、事件驱动、工具注册、安全沙箱。扁平化设计，色彩明亮活泼，清晰的现代科技风格 --ar 16:9 --v 6

## 第 3 页：核心能力 (Pipeline)

**页面主题**: 七阶段智能流水线

**视觉风格要求**:
- 环形或弧形布局展示 7 个阶段
- 阶段之间用箭头连接
- 中心留白，简洁大气

**核心视觉元素**:
1. **环形阶段展示**: 7 个彩色圆形节点组成环形/弧形
   - 💡 IDEA (青色 #06B6D4)
   - 📋 PRD (紫色 #8B5CF6)
   - 🎨 DESIGN (粉色 #EC4899)
   - 📝 PLAN (橙色 #F59E0B)
   - ⚙️ CODING (绿色 #10B981)
   - 🔍 CHECK (红色 #F43F5E)
   - 🚀 DELIVERY (蓝色 #3B82F6)

2. **核心机制面板** (右侧)
   - 🎭 Actor 生成内容
   - 🔎 Critic 评估质量
   - 🔁 迭代优化收敛
   - ⏳ 人工确认节点

3. **技术栈支持** (右下角)
   - React, Vue, Next.js, Rust, Python, Node.js, Go, Tauri

4. **自动化能力** (右侧)
   - 依赖安装, 代码编译, 单元测试, Lint检查, 类型检查, 部署打包

5. **6 个特色功能卡片** (底部)
   - Actor-Critic 双智能体
   - Human-in-Loop 审核
   - 阶段可跳转
   - 全量质量检查
   - 迭代状态持久化
   - 多技术栈支持

**布局要求**:
- 左侧 2/3: 环形阶段图 + 底部功能卡片
- 右侧 1/3: 核心机制 + 技术栈 + 自动化能力

## 通用提示词模板

如果你需要调整或生成其他风格，可以使用以下模板：

```
现代商业演示幻灯片，16:9宽屏，[背景描述]。[主要视觉元素描述]。[配色要求]。[风格要求]如扁平化设计、渐变色、简洁专业等]。--ar 16:9 --v 6
```

### 常用参数说明

| 参数 | 说明 |
|------|------|
| `--ar 16:9` | 宽屏比例 |
| `--ar 9:16` | 竖屏比例 |
| `--v 6` | Midjourney V6 版本 |
| `--nijis 5` | 动漫风格 |
| `--style raw` | 更真实的风格 |

## 注意事项

1. **分辨率**: 生成的图片可能需要根据实际用途调整分辨率，建议使用 1920x1080 或更高
2. **文字**: AI 生图可能无法准确生成文字，建议将生成的图片作为背景或参考，在 PPT/Keynote 中叠加文字
3. **元素位置**: 可以先生成整体布局，再针对每个区域生成细节
4. **版权**: 确认所选 AI 模型的商业使用条款
