# System Boundary Interface Documentation

This document describes the system's external invocation interfaces, including CLI commands, API endpoints, configuration parameters, and other boundary mechanisms.

## Command Line Interface (CLI)

### cowork

**Description**: AI驱动的软件开发系统主命令

**Source File**: `crates/cowork-cli/src/main.rs`

**Arguments**:

- `command` (enum): required - 要执行的子命令

**Options**:

- `config, c`(string): optional - 配置文件路径（默认：config.toml） (default: `config.toml`)
- `verbose, v`(boolean): optional - 启用详细日志记录 (default: `false`)
- `stream, s`(boolean): optional - 启用LLM流式输出（实时显示AI思考过程） (default: `false`)

**Usage Examples**:

```bash
cowork new "创建一个在线考试系统"
```

```bash
cowork resume --base session-1234567890
```

```bash
cowork modify "添加用户注册功能" --stream
```

```bash
cowork status --sessions
```

```bash
cowork revert --from coding
```

### cowork resume

**Description**: 恢复现有项目

**Source File**: `crates/cowork-cli/src/main.rs`

- `base, b`(string): optional - 从特定会话ID恢复（可选，默认为最新成功会话）

```bash

```

## Integration Suggestions

### IDE插件集成

开发VS Code或其他IDE插件，提供图形化界面和实时项目状态监控

**Example Code**:

```
// VS Code扩展示例
const vscode = require('vscode');
class CoworkForgeProvider {
  provideTasks() {
    return [
      new vscode.Task(
        { type: 'cowork' }, 
        vscode.TaskScope.Workspace,
        'New Project',
        'cowork',
        new vscode.ShellExecution('cowork new "项目描述"')
      )
    ];
  }
}
```

**Best Practices**:

- 提供项目模板库支持
- 集成版本控制状态显示
- 实现实时进度监控
- 支持一键回退到任意阶段

### CI/CD流水线集成

将Cowork Forge集成到持续集成流程中，自动化代码审查和部署

**Example Code**:

```
# GitHub Actions示例
name: Cowork Forge CI
on: [push, pull_request]
jobs:
  ai-review:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Setup Cowork Forge
        run: |
          cargo install --path .
          cowork modify "CI自动化优化" --stream
      - name: Run Tests
        run: cargo test
```

**Best Practices**:

- 在代码合并前进行AI辅助审查
- 自动化生成变更报告
- 集成质量门禁检查
- 支持回滚到稳定版本

### 项目管理工具集成

与Jira、Trello等项目管理工具集成，同步任务状态和进度

**Example Code**:

```
// Jira集成示例
const jiraClient = new JiraApi({
  host: 'company.atlassian.net',
  authentication: {
    username: process.env.JIRA_USER,
    password: process.env.JIRA_TOKEN
  }
});

async function syncTasks(sessionId) {
  const plan = await getPlan(sessionId);
  for (const task of plan.tasks) {
    await jiraClient.addNewIssue({
      fields: {
        project: { key: 'COWORK' },
        summary: task.title,
        description: task.description
      }
    });
  }
}
```

**Best Practices**:

- 双向同步任务状态
- 自动化需求跟踪
- 集成进度报告生成
- 支持敏捷开发流程
